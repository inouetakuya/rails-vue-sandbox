rails-vue-sandbox
=================

【動画付き】Rails 5.1 で作る Vue.js アプリケーション ～ Heroku デプロイからシステムテストまで～ - Qiita
https://qiita.com/jnchito/items/30ab14ebf29b945559f6

## Requirements

* Ruby 2.4.2
* ChromeDriver

## Setup

```
bundle install
```

## Development

```
bundle exec foreman start
```

## Testing

```
bin/rake test
bin/rake test:system
```
